# Healthcare Analysis Dashboard (Power BI)

##  Project Overview
This project presents a **Healthcare Analysis Dashboard** built using Power BI to analyze screening eligibility, participation rates, and patient demographics.

The dashboard helps healthcare organizations:
- Monitor screening performance
- Understand patient demographics
- Improve outreach and healthcare delivery

## Key Metrics
- **Members Eligible for Screening:** 6,597  
-  **Members Screened:** 4,184 (63%)  
- **Members Not Screened:** 2,413 (37%)  

---

## Dashboard Features

### 🔹 Screening Analysis
- Total eligible members
- Screening completion rate
- Non-screened population insights

### 🔹 Demographic Insights
- Members by **ethnicity**
- Members by **region** (USCA, APAC, EMEA, Others)
- Members by **race**
- Members by **preferred language**

### 🔹 Filters & Interactivity
- Region filter  
- Ethnicity filter  
- Race filter  
- Metric selection panel
